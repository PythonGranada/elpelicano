Sobre la web

lun 25 junio 2018 in Información

Hola! Hemos cambiado la manera en la que funciona nuestra web. Anteriormente usabamos Django, con la idea de poder crear extensiones y fomentar la participación.

Finalmente esto se volvió una carga más que una ventaja y hemos decidido crear este nuevo sitio usando Pelican.

El sitio se encuentra público en github en este repositorio.

Muchas gracias a todos, nos vemos proximamente.
